# Delete File

Delete a file from Cocobase storage.

## Endpoint

```
DELETE /storage/files/{id}
```

## Parameters

| Parameter | Type | Location | Required | Description |
|-----------|------|----------|----------|-------------|
| `id` | string | path | Yes | File ID |

## Response

```json
{
  "success": true,
  "data": {
    "id": "file_abc123",
    "deleted": true
  }
}
```

## Examples

### cURL

```bash
curl -X DELETE https://api.cocobase.buzz/storage/files/file_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### JavaScript

```javascript
const response = await fetch(
  "https://api.cocobase.buzz/storage/files/file_abc123",
  {
    method: "DELETE",
    headers: {
      Authorization: "Bearer YOUR_API_KEY",
    },
  }
);

const result = await response.json();
console.log(result.data.deleted); // true
```

### Python

```python
import requests

response = requests.delete(
    "https://api.cocobase.buzz/storage/files/file_abc123",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
)

result = response.json()
print(result["data"]["deleted"])  # True
```

## Batch Delete

Delete multiple files at once.

### Endpoint

```
POST /storage/files/batch-delete
```

### Request Body

```json
{
  "ids": ["file_abc123", "file_def456", "file_ghi789"]
}
```

### Response

```json
{
  "success": true,
  "data": {
    "deleted": 3,
    "ids": ["file_abc123", "file_def456", "file_ghi789"]
  }
}
```

### Example

```bash
curl -X POST https://api.cocobase.buzz/storage/files/batch-delete \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "ids": ["file_abc123", "file_def456"]
  }'
```

## Error Responses

### File Not Found

```json
{
  "success": false,
  "error": {
    "code": "NOT_FOUND",
    "message": "File not found"
  }
}
```

### Permission Denied

```json
{
  "success": false,
  "error": {
    "code": "FORBIDDEN",
    "message": "You do not have permission to delete this file"
  }
}
```
