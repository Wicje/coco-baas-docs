# Update Document

Update an existing document in a collection.

## Endpoint

```
PATCH /collections/{collection}/{id}
```

## Parameters

| Parameter | Type | Location | Required | Description |
|-----------|------|----------|----------|-------------|
| `collection` | string | path | Yes | Collection name |
| `id` | string | path | Yes | Document ID |

## Request Body

JSON object with fields to update. Only specified fields are updated.

```json
{
  "title": "Updated Title",
  "published": false
}
```

## Response

```json
{
  "success": true,
  "data": {
    "id": "doc_abc123",
    "collection": "posts",
    "data": {
      "title": "Updated Title",
      "content": "Original content",
      "published": false
    },
    "createdAt": "2024-01-15T10:30:00Z",
    "updatedAt": "2024-01-15T11:00:00Z"
  }
}
```

## Examples

### cURL

```bash
curl -X PATCH https://api.cocobase.buzz/collections/posts/doc_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Updated Title",
    "published": false
  }'
```

### JavaScript

```javascript
const response = await fetch(
  "https://api.cocobase.buzz/collections/posts/doc_abc123",
  {
    method: "PATCH",
    headers: {
      Authorization: "Bearer YOUR_API_KEY",
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      title: "Updated Title",
      published: false,
    }),
  }
);

const result = await response.json();
console.log(result.data);
```

### Python

```python
import requests

response = requests.patch(
    "https://api.cocobase.buzz/collections/posts/doc_abc123",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json",
    },
    json={
        "title": "Updated Title",
        "published": False,
    },
)

result = response.json()
print(result["data"])
```

## Full Replace (PUT)

To replace the entire document, use PUT instead of PATCH:

```
PUT /collections/{collection}/{id}
```

```bash
curl -X PUT https://api.cocobase.buzz/collections/posts/doc_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "New Title",
    "content": "New Content",
    "published": true
  }'
```

## Error Responses

### Document Not Found

```json
{
  "success": false,
  "error": {
    "code": "NOT_FOUND",
    "message": "Document not found"
  }
}
```

### Invalid Request Body

```json
{
  "success": false,
  "error": {
    "code": "INVALID_BODY",
    "message": "Request body must be valid JSON"
  }
}
```
