# Go SDK

Complete guide to using Cocobase with Go applications.

## Installation

```bash
go get github.com/cocobase-team/cocobase-go
```

## Quick Start

### Initialize Cocobase

```go
package main

import (
    "github.com/cocobase-team/cocobase-go"
)

func main() {
    db := cocobase.NewClient(cocobase.Config{
        APIKey:  "YOUR_API_KEY",
        BaseURL: "https://api.cocobase.buzz", // Optional
    })
}
```

### Basic Operations

```go
// List documents
docs, err := db.ListDocuments("posts", nil)
if err != nil {
    log.Fatal(err)
}

// Get single document
doc, err := db.GetDocument("posts", "doc-id")
if err != nil {
    log.Fatal(err)
}

// Create document
created, err := db.CreateDocument("posts", map[string]interface{}{
    "title":   "My First Post",
    "content": "Hello World!",
})

// Update document
err = db.UpdateDocument("posts", "doc-id", map[string]interface{}{
    "title": "Updated Title",
})

// Delete document
err = db.DeleteDocument("posts", "doc-id")
```

## Querying Data

### Simple Filters

```go
// Find active users older than 18
users, err := db.ListDocuments("users", &cocobase.QueryOptions{
    Filters: map[string]interface{}{
        "status":  "active",
        "age__gt": 18,
    },
})
```

### Query Builder

```go
query := cocobase.NewQueryBuilder().
    Where("status", "active").
    WhereGreaterThan("age", 18).
    WhereContains("email", "@gmail.com").
    OrderByDesc("createdAt").
    Limit(10)

users, err := db.ListDocuments("users", &cocobase.QueryOptions{
    Query: query,
})
```

## Operators Reference

### Comparison Operators

| Operator | Filter Key | Example |
|----------|-----------|---------|
| Equal | `field` | `"status": "active"` |
| Greater Than | `field__gt` | `"age__gt": 18` |
| Greater or Equal | `field__gte` | `"age__gte": 18` |
| Less Than | `field__lt` | `"age__lt": 65` |
| Less or Equal | `field__lte` | `"age__lte": 65` |
| Not Equal | `field__ne` | `"status__ne": "deleted"` |

### String Operators

| Operator | Filter Key | Example |
|----------|-----------|---------|
| Contains | `field__contains` | `"title__contains": "go"` |
| Starts With | `field__startswith` | `"email__startswith": "admin"` |
| Ends With | `field__endswith` | `"domain__endswith": ".com"` |

### Array Operators

| Operator | Filter Key | Example |
|----------|-----------|---------|
| In Array | `field__in` | `"status__in": "active,pending"` |
| Not In Array | `field__notin` | `"status__notin": "deleted,archived"` |

## Sorting and Pagination

### Sorting

```go
docs, err := db.ListDocuments("posts", &cocobase.QueryOptions{
    Filters: map[string]interface{}{
        "orderBy": "createdAt",
        "order":   "desc",
    },
})
```

### Pagination

```go
// Get page 3 (20 items per page)
docs, err := db.ListDocuments("posts", &cocobase.QueryOptions{
    Filters: map[string]interface{}{
        "limit":  20,
        "offset": 40,
    },
})
```

## Working with Structs

### Define Your Types

```go
type Book struct {
    Title       string  `json:"title"`
    Author      string  `json:"author"`
    Price       float64 `json:"price"`
    PublishedAt string  `json:"publishedAt,omitempty"`
}
```

### Create with Structs

```go
newBook := Book{
    Title:  "Clean Code",
    Author: "Robert Martin",
    Price:  45.99,
}

created, err := db.CreateDocument("books", newBook)
```

### Parse Response to Struct

```go
doc, err := db.GetDocument("books", "doc-id")
if err != nil {
    log.Fatal(err)
}

var book Book
err = doc.DataAs(&book)
if err != nil {
    log.Fatal(err)
}

fmt.Println(book.Title)
```

## Authentication

### Email/Password

```go
// Register
err := db.Auth.Register(cocobase.AuthCredentials{
    Email:    "user@example.com",
    Password: "securePassword123",
})

// Get user after registration
user, err := db.Auth.GetUser()
fmt.Println("User ID:", user.ID)

// Login
err = db.Auth.Login(cocobase.AuthCredentials{
    Email:    "user@example.com",
    Password: "securePassword123",
})

// Get user after login
user, err = db.Auth.GetUser()

// Logout
err = db.Auth.Logout()

// Check current user
currentUser := db.Auth.CurrentUser()
```

### OAuth

```go
// Google OAuth
err := db.Auth.SignInWithGoogle()
user, err := db.Auth.GetUser()

// GitHub OAuth
err = db.Auth.SignInWithGitHub()
user, err = db.Auth.GetUser()
```

## Real-time Data

### Watch Collection

```go
// Watch for changes
cancel, err := db.WatchCollection("posts", func(event cocobase.Event) {
    fmt.Println("Event type:", event.Type)
    fmt.Println("Document:", event.Document)
})

// Stop watching
cancel()
```

### Watch Document

```go
cancel, err := db.WatchDocument("posts", "doc-id", func(doc cocobase.Document) {
    fmt.Println("Document updated:", doc.Data)
})

// Stop watching
cancel()
```

## Error Handling

```go
doc, err := db.GetDocument("posts", "non-existent-id")
if err != nil {
    if cocobase.IsNotFound(err) {
        fmt.Println("Document not found")
    } else if cocobase.IsUnauthorized(err) {
        fmt.Println("Not authorized")
    } else {
        fmt.Println("Error:", err)
    }
}
```

## Best Practices

### 1. Use Structs for Type Safety

```go
// Good
type Post struct {
    Title   string `json:"title"`
    Content string `json:"content"`
}

var post Post
doc.DataAs(&post)

// Avoid
data := doc.Data.(map[string]interface{})
```

### 2. Always Handle Errors

```go
doc, err := db.GetDocument("posts", id)
if err != nil {
    return nil, fmt.Errorf("failed to get post: %w", err)
}
```

### 3. Use Environment Variables

```go
db := cocobase.NewClient(cocobase.Config{
    APIKey: os.Getenv("COCOBASE_API_KEY"),
})
```

### 4. Always Set Limits

```go
// Good
docs, err := db.ListDocuments("posts", &cocobase.QueryOptions{
    Filters: map[string]interface{}{"limit": 20},
})

// Bad - could return thousands
docs, err := db.ListDocuments("posts", nil)
```
